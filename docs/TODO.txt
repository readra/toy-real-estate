1. 시/구/동 기준 리스트 박스 추가(시/구 모두 선택해야 함) (완료)
1.1. 오픈 API 제약사항으로 시/구 까지는 있어야 하고, 동까지 선택하면 백엔드에서 필터링 처리 (완료)
2. 매매 가격, 거래 일자 범위 추가 (완료)

2.2. 거래 일자 달력 형태 시작/종료 추가 (완료)
2.3. 거래 일자 달력 버튼 클릭 위치에 별도 창으로 출력 (진행중, before/after 유효성 검증 필요)
2.4. 년/월/일 클릭해서 달력 확장 (진행중, 년 고르고 월 고르고 일 고르고)
